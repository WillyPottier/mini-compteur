# Mini Compteur

Petit compteur +/– en HTML/CSS/JS.
- Limite basse 0
- Limite haute 20
- Cube visible à partir de 5, orange entre 5 et 10, puis couleur par défaut
- Le cube grossit avec la valeur (transform: scale)

## Lancer
Ouvrir `index.html` dans un navigateur.
